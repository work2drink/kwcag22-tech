<Aside title="검사 항목">
자동으로 소리가 재생되지 않아야 한다.
</Aside>

## 목적
웹 페이지에서 사용자의 의도와 관계없이 자동으로 소리가 재생되는 것을 방지하여, 소리가 사용자의 웹 페이지 이용을 방해하지 않도록 한다. 자동 재생되는 소리는 사용자의 집중력을 저해하거나, 보조 기술을 사용하는 사용자에게 불편을 초래할 수 있다.

##	배경 및 필요성
자동 재생되는 소리가 화면낭독프로그램의 음성과 동시에 재생되면, 사용자는 웹페이지의 음성 정보와 겹쳐서 제대로 이해하기 어렵다. 특히, 화면낭독프로그램의 음성과 재생되는 오디오가 동일한 볼륨 수준이거나 같은 출력 장치를 공유할 경우, 정보 전달이 방해되어 문제가 더욱 심각해질 수 있다.

또한, 청각에 제한이 있는 사용자나 인지 능력이 낮은 사용자는 예기치 않은 소리 재생으로 인해 혼란을 겪거나 곤란한 상황이 발생할 수 있다.

이와 같이 자동 재생되는 소리는 사용자에게 불필요한 방해 요소가 될 수 있으므로, 기본적으로 자동 재생을 비활성화하고, 사용자가 원할 때 직접 제어할 수 있도록 제공해야 한다.

##	기본 규칙
- 웹 페이지에서는 기본적으로 소리가 자동으로 재생되는 콘텐츠를 제공하지 않는다.
- 자동 재생되는 소리를 불가피하게 제공해야 하는 경우, 해당 소리는 3초 이내에 자동으로 정지되거나, 사용자가 즉시 정지 또는 음소거할 수 있는 기능을 제공해야 한다.
- 사용자가 즉시 정지하는 방법은 키보드만으로도 쉽게 조작할 수 있어야 하며, 다양한 사용자 환경에서도 동작해야 한다. 예를 들어, 키보드 단축키(ESC 키, 혹은 사용자 정의 가능 단축키)를 제공하거나, 문서의 시작 부분에 자동 재생되는 소리를 제어할 수 있는 버튼이나 메뉴를 배치해야 한다.

## 적용 사례
### 1)	정지 상태로 제공된 동영상
화면낭독프로그램 사용자 등은 자동 재생되는 오디오로 인해 콘텐츠 탐색이나 조작이 어려워질 수 있으므로, 기본적으로 정지 상태로 제공하고 사용자가 직접 재생할 수 있도록 해야 한다.
다음은 소식 페이지 내 동영상이나 음원 콘텐츠가 초기에는 멈춤 상태로 제공되며, 사용자가 원하는 시점에 재생할 수 있도록 하였다.

![이미지. 페이지 내 동영상이 멈춤 상태로 제공된 사례 (출처: 카카오)](/images/image089.png)

![이미지. 선택한 음원 콘텐츠를 멈춤 상태로 제공한 사례 (출처: KT 회사소개)](/images/image090.png)

### 2)	지정된 단축키로 자동 재생 제어
페이지 접근 시 자동 재생되는 배경음을 ESC 키를 눌러 즉시 멈출 수 있도록 제공한 사례이다.
기능 정지를 위해 자주 사용하는 ESC 키를 활용하여 소리를 멈출 수 있는 기능을 제공하면, 화면낭독프로그램 사용자가 배경음을 빠르게 끌 수 있으며, 안내 음성과 중복되지 않고 원활하게 콘텐츠를 탐색할 수 있도록 도와준다.

![이미지. 페이지 접근 시 자동으로 배경음이 재생되는 예시](/images/image091.png)

### 3)	음소거 상태로 자동 재생되는 콘텐츠
3초 이상 자동 재생되는 콘텐츠는 음소거 상태로 제공되며, 사용자가 필요할 때 소리를 켤 수 있는 버튼을 함께 제공해야 한다.
이러한 방식은 화면낭독프로그램 사용자가 웹 페이지를 탐색할 때 예기치 않은 소리로 인한 방해를 예방할 수 있다. 또한, 사용자가 직접 소리 재생 여부를 결정할 수 있어 더 나은 사용자 경험을 제공한다. 공공장소나 조용한 환경에서 웹사이트를 이용하는 사용자에게도 유용하다.

![이미지. 메인페이지 영상이 음소거 상태로 제공된 사례 (출처: 카카오페이)](/images/image092.png)

### 4)	재생 시간이 3초 미만인 소리 사용
자동 재생되는 콘텐츠를 제공하는 경우, 소리가 3초 미만으로 재생되도록 설정한다. 3초 이후에는 음소거 상태로 전환하거나 자동으로 재생이 멈추도록 해야 한다. 또한, 이후 소리가 반복되지 않도록 설정 확인이 필요하다.

![이미지. 3초 미만의 효과음이 자동 재생되는 예시](/images/image093.png)

## 개선 필요 사례

### 1)	자동으로 재생되는 배경 음악
다음은 웹툰에서 특정 회차에 진입하면 배경 음악이 자동으로 재생되는 사례이다.

배경 음악을 제어할 수 있는 기능이 제공되었으나, 제어 버튼이 본문 영역에 위치하여 접근이 어렵고, ESC 키도 작동하지 않아 즉시 제어할 방법이 없다. 또한, 화면낭독프로그램 사용자는 음악을 멈추거나 음소거할 방법을 즉시 찾기 어려워 불편을 겪을 수 있다.

![이미지. 배경 음악이 자동으로 재생 페이지와 정지 버튼이 본문에 제공 사례](/images/image094.png)

#### 개선 방법
페이지 진입 시 특정 키(예: ESC)를 눌러 배경 음악을 즉시 음소거하거나 멈출 수 있는 기능을 제공해야 하며, 일반적으로 ESC 키는 음소거 또는 재생 중단 기능에 사용되므로, 이를 활용할 수 있도록 한다. 

또한, 페이지 진입 시 배경 음악 재생 여부를 사용자가 사전에 선택할 수 있도록 하는 옵션을 제공하는 것도 대안이 될 수 있다.

### 2)	키보드 초점만으로 자동 재생되는 오디오
버튼이 키보드 초점을 얻는 것만으로 자동으로 오디오가 재생되는 사례이다.

화면낭독프로그램 사용자는 오디오와 화면낭독프로그램의 음성이 겹쳐 탐색 중이던 정보를 놓칠 가능성이 있다. 또한 키보드만 사용하는 사용자의 경우 원치 않는 오디오가 반복적으로 재생될 수 있어 탐색 경험을 방해하고 접근성을 저해할 수 있다. 이처럼 사용자가 오디오를 제어하는 방법이 명확하지 않으면, 오디오를 멈추거나 음소거하기 어려워질 수 있다.

![이미지. 버튼이 초점을 얻는 것만으로 음악이 재생되는 예시](/images/image095.png)

#### 개선 방법
오디오는 초점을 얻는 것만으로 자동 재생되지 않도록 설정하고, 필요하다면 사용자가 직접 재생 여부를 선택할 수 있도록 해야 한다.

### 3)	자동으로 재생되는 소리가 포함된 멀티미디어 콘텐츠
다음은 페이지 진입 시, 소리가 포함된 동영상 콘텐츠가 자동으로 재생되는 사례이다.

음식 레시피 페이지 내 본문 최하단에 있는 동영상 콘텐츠가 자동 재생되지만, 이를 제어할 수 있는 기능이 문서 시작 부분에 제공되지 않는다.

이에 따라 사용자는 소리가 나는 위치를 확인하고 끄기 위해 페이지의 마지막 부분까지 탐색해야 하는 불편을 겪는다. 또한, 화면낭독프로그램 사용자의 경우 자동 재생되는 소리가 화면 낭독 음성과 겹쳐 콘텐츠 탐색이 매우 어려워진다.

![이미지. 페이지에 접속 시 바로 재생되는 멀티미디어 콘텐츠 사례](/images/image096.png)

#### 개선 방법
멀티미디어 콘텐츠가 자동으로 재생되지 않도록 제공하거나, 오디오 콘텐츠가 3초 이내에 자동 정지되거나, 사용자가 즉시 정지하거나 음소거할 수 있는 제어 기능을 추가한다.

### 4)	키보드 초점만으로 자동 재생되는 오디오
버튼이 키보드 초점을 얻는 것만으로 자동으로 오디오가 재생되는 사례이다.

화면낭독프로그램 사용자는 오디오와 화면낭독프로그램의 음성이 겹쳐 탐색 중이던 정보를 놓칠 가능성이 있다. 또한 키보드만 사용하는 사용자의 경우 원치 않는 오디오가 반복적으로 재생될 수 있어 탐색 경험을 방해하고 접근성을 저해할 수 있다.

이처럼 사용자가 오디오를 제어하는 방법이 명확하지 않으면, 오디오를 멈추거나 음소거하기 어려워질 수 있다.

![이미지. 버튼이 초점을 얻는 것만으로 음악이 재생되는 예시](/images/image095.png)

#### 개선 방법
오디오는 초점을 얻는 것만으로 자동 재생되지 않도록 설정하고, 필요하다면 사용자가 직접 재생 여부를 선택할 수 있도록 해야 한다.

5)	3초를 초과하여 자동 재생되는 오디오 콘텐츠
다음은 게임 사이트의 로그인 페이지에 진입 시 3초를 초과하는 오디오 콘텐츠가 자동으로 재생되는 사례이다.

게임 캐릭터의 대사가 자동으로 흘러나오지만, 사용자가 이를 제어하거나 중단할 수 있는 기능이 제공되지 않는다. 이에 따라 화면낭독프로그램 사용자는 화면 낭독 소리와 겹쳐 정보 전달을 방해받을 수 있다.

![이미지. 캐릭터의 대사가 자동 재생되는 게임 사이트 예시](/images/image097.png)

#### 개선 방법
특정 키(예: ESC 키)를 눌러 캐릭터의 대사를 즉시 음소거하거나 멈출 수 있는 기능을 제공하거나, 자동 재생되는 콘텐츠를 제어할 수 있는 버튼을 제공해야 한다.
