<Aside title="검사 항목">
반복되는 입력 정보는 자동 입력 또는 선택 입력할 수 있어야 한다.
</Aside>

## 목적
인지 능력 또는 기억력에 제약이 있는 사용자나 보조 입력 도구를 사용하는 사용자가 여러 단계에 걸쳐 정보를 반복 입력하는 과정에서 느끼는 부담을 줄이고, 원활한 서비스 이용을 돕기 위함이다.

##	배경 및 필요성
여러 단계에 걸쳐 정보를 입력할 때 동일한 내용을 반복해서 입력하도록 요구하면, 모든 사용자에게 번거로울 뿐만 아니라 인지적 부담이 큰 사용자는 중도에 입력을 중단하거나 실수할 가능성이 높다. 또한, 음성 명령이나 한 손 입력 등 다양한 대체 입력 방식을 사용할 때도 반복 입력은 시간과 노력을 매우 증가시킨다.

반복 입력을 최소화하면 사용자의 인지 부담을 줄이고 입력 과정을 단순화하여 사용자 만족도를 높일 수 있다. 이를 위해 이미 입력한 정보를 재사용하거나 자동 완성 기능을 제공하며, 불필요한 반복 입력 요청을 제거하는 방식을 적용한다.

따라서, 반복 입력으로 인한 문제는 사용자가 이미 입력한 정보를 재사용하거나, 자동 입력/선택 기능을 통해 불필요한 반복을 줄임으로써 해결할 수 있다.

단, 비밀번호와 같이 보안을 위해 재입력이 필요한 경우나 이전에 입력한 정보가 유효하지 않은 경우는 예외로 한다.

##	기본 규칙
- 하나의 프로세스 안에서 사용자가 이미 입력한 정보를 재사용하여 반복 입력을 방지한다.
- 이전에 입력한 정보를 자동으로 채워주거나, 이전 입력 정보 중에서 선택하여 입력할 수 있는 기능을 제공한다.
- 동일한 정보를 여러 번 입력하도록 요구하는 단계를 제거한다.
- 단, 브라우저의 기본 자동 완성 기능(autocomplete)만으로는 부족하므로, 별도의 기능으로 구현해야 한다.

<Card title="참고. 브라우저의 자동 완성(autocomplete) 기능" icon="open-book">
브라우저의 자동 완성(autocomplete) 기능은 반복 입력 문제를 완전히 해결하지 못한다. 이 기능은 과거에 입력된 데이터에 의존하기 때문에 실수로 잘못 입력한 정보이거나 오래되었을 수 있으며, 웹사이트 또는 입력 중인 프로세스와 직접적인 연계가 없을 수 있다. 이런 이유 등으로 단순히 브라우저의 자동 완성 기능에 의존하는 것만으로는 충분하지 않다. 따라서, 웹사이트 자체적으로 반복 입력된 정보를 제공하거나, 동일한 정보를 다시 묻지 않는 방식으로 설계하는 것이 필요하다.

## 적용 사례
### 1)	이전에 입력한 정보를 자동으로 채우는 기능 제공
상품 주문 과정에서 주문자와 배송지가 동일한 경우, '주문자 정보와 동일' 체크 상자를 제공하여 이름, 이메일, 휴대폰 번호 등 기존에 입력한 정보를 자동으로 배송 정보에 반영할 수 있도록 한다.

	![](/images/image261.png)

### 2)	동일한 프로세스 내에서 이전에 선택한 정보를 자동으로 채워서 제공
결제 단계에서 선택한 카드 결제 무이자 할부 정보가 외부 결제 페이지로 이동할 때도 자동으로 유지되어, 사용자가 다시 선택할 필요가 없도록 한 사례이다. 만약 할부 정보를 다시 직접 입력해야 했다면, 사용자는 이전에 선택한 정보와 다르게 설정하여 결제 프로세스를 일부 다시 진행해야 하거나, 결제가 실패하는 상황을 겪을 수도 있다.

## 개선 필요 사례
### 1)	동일한 정보를 재사용 할 수 있는 기능 미제공
다음은 쇼핑몰에서 주문서를 작성하는 동안에 동일한 정보를 반복 입력해야 하는 사례이다.

앞서 입력한 정보를 재사용하는 기능이 제공되지 않으면, 주문자 정보와 받는 사람의 정보가 동일할 때도 반복해서 입력해야 한다. 이 경우, 대체 입력 수단을 사용하는 사용자는 입력 과정에 많은 시간과 노력을 들이거나 잘못 입력하여 실수가 발생할 가능성이 높다.

	<figcaption>이미지. 주문자 정보와 배송 정보를 중복으로 입력해야 하는 예시</figcaption>

#### 개선 방법
주문자와 받는 사람이 동일할 경우, 같은 정보를 반복해서 입력하지 않고 자동으로 채울 수 있는 기능을 제공한다. 예를 들어, '주문자와 동일' 체크 상자를 제공하여 사용자가 선택하면 자동으로 정보를 채우도록 한다.

### 2)	중복 입력 정보의 자동 입력 미제공
다음은 검색 결과 페이지에서 검색어 입력란에 이전에 입력한 검색어를 확인할 수 없는 사례이다.

검색 결과 페이지에서 사용자가 이전에 입력한 검색어가 유지되지 않아, 다시 검색하거나 검색어를 수정하려면 처음부터 재입력해야 한다.

이는 특히 검색어를 다시 떠올리고 입력하는 과정에서 인지력이나 기억력에 어려움이 있는 사용자들에게 부담을 주며, 대체 입력 수단을 사용하는 사용자나 음성 명령 사용자들에게는 입력 작업에 시간과 노력이 필요하게 된다.

### 개선 방법
검색 결과 페이지에서 검색 입력란 또는 검색 결과에 사용자가 입력한 키워드를 표시하여, 이전 검색어를 쉽게 재사용할 수 있도록 한다.
